apa itu javascript

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat internet bekerja lebih dinamis dan interaktif. Dikembangkan oleh Brendan Eich pada tahun 1995, JavaScript adalah bahasa pemrograman yang ringan, bersifat dinamis, dan dapat diarahkan untuk oriented programming maupun functional programming 1 . JavaScript umumnya digunakan untuk melakukan perubahan tampilan website, seperti ketika mengklik tombol, menentukan layout, ngasih efek dinamik atau animasi, dan membuat aplikasi yang berhubungan dengan database
